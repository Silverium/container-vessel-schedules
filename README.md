## Dev mode
Install node modules
```bash
npm i
```
Build the app
```bash
npm run build
```
Execute it with node
```bash
node dist/cli
```
Or on the fly with `ts-node`
```bash
ts-node source/cli
```
## Test
Run
```bash
npm run test
```
## Contribute
Install node modules and then run the dev mode with
```bash
npm run dev
```
And it will watch the files at `./source` so you can execute the app any time.
